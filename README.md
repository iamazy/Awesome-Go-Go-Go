Awesome Go Go Go
========================
这个项目收集了一些比较简短，代码行数不太大，适合Go语言新手的童鞋练手的项目，其中有一些项目非常的有趣（如JSON组合中前五个项目），用简练的算法实现强大的功能，帮助童鞋们加深对Go语言的理解。
